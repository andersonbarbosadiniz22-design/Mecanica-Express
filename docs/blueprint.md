# **App Name**: Mecânica Express

## Core Features:

- Product Content Presentation: Display all textual content (Hero, Problem, Solution, Learnings, Testimonials, Offer, Guarantee, CTAs, Footer) in Portuguese (Brazil) with a compelling and persuasive flow.
- SEO & Meta Implementation: Integrate the provided Meta Title, Meta Description, and Keywords. Implement a structured HTML layout with H1, H2, H3 headings for optimal search engine visibility and accessibility.
- Mobile-First Responsive Design: Ensure the entire landing page is designed and optimized for mobile devices first, providing a seamless and high-quality user experience across all screen sizes.
- Interactive Call-to-Actions (CTAs): Implement prominent and highly visible CTA buttons ('QUERO ACESSO AGORA', 'BAIXAR AGORA') positioned strategically, including 'above the fold', to drive immediate conversions.
- Dynamic Image Gallery: Display relevant product images, engine diagrams, mechanical systems, and apostila previews in an attractive and responsive grid gallery format.
- Social Proof Display: Showcase customer testimonials with ratings in a dedicated, visually appealing section to build trust and credibility with potential buyers.
- AI Sales Assistant Tool: An integrated AI-powered chat tool that uses generative AI to provide instant answers to prospective customers' questions about the automotive mechanics package, helping to clarify doubts and guide purchase decisions in Portuguese.

## Style Guidelines:

- The dominant color scheme is dark. Primary interactive elements and calls to action will use a striking red (#c40000, HSL: 0, 100%, 38%) to create urgency and visual focal points against a deep black background (#111111, HSL: 0, 0%, 7%). A metallic gray (#5a5a5a, HSL: 0, 0%, 35%) will be used for secondary elements and design accents, contributing to the professional, industrial aesthetic. White (#ffffff) will provide clean contrast for body text and important information.
- Headlines and prominent titles will utilize 'Space Grotesk' (sans-serif) for its modern, technical, and slightly futuristic appeal, aligning with the automotive and industrial theme. Body text and longer content blocks will use 'Inter' (sans-serif) for its clarity, objectivity, and excellent readability across various screen sizes, ensuring a professional and accessible experience.
- Use clean, minimalist line-art icons that visually represent mechanical concepts like gears, wrenches, engine diagrams, and automotive tools. Icons should have consistent stroke weights and adhere to the black, gray, and red color palette to reinforce the industrial mechanical look and theme.
- A mobile-first responsive layout with a clear hierarchical structure (H1, H2, H3) and ample negative space to ensure content readability. The Hero section will prominently feature a compelling headline and CTA 'above the fold'. Content will be segmented into distinct sections as requested, utilizing grid layouts for the image gallery and structured boxes for testimonials and offers to guide the user visually through the sales funnel. Conversion-focused elements are strategically placed to maximize impact.
- Subtle, performant animations will be employed to enhance user interaction without compromising load times. This includes smooth scroll-reveal effects for sections, tasteful hover states for CTA buttons and interactive elements, and minor micro-interactions to provide visual feedback and create a polished, engaging experience.
